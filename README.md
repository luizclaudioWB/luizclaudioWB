## O que estou construindo

🏥 **Médica de Bolso** — plataforma B2C de telemedicina em produção. Java 21, Quarkus 3.24, PostgreSQL, AWS (EC2/RDS/S3/SES), MercadoPago Bricks v2, WebSocket com OTP por sessão, LGPD. **Concebi, desenvolvi e mantenho sozinho — back, front, infra e deploy.**

⚡ **Wisebyte (CLT)** — backend Java + Quarkus + AWS pra clientes de energia e mineração. **OCR em produção em 5 distribuidoras** (Energisa, CPFL, EDP, Neoenergia, Light) usando AWS Textract com detecção de layout via Jaro-Winkler + Levenshtein normalizado.

🤖 **Migrando pra GenAI aplicada** — Python (asyncio, Pydantic v2, FastAPI), RAG com pgvector, agents (ReAct, planning, guardrails) e protocolo MCP. **Próximo passo declarado:** integrar agente de pré-triagem médica ao Médica de Bolso.
